# 🎵 Top Song Duration Analysis

This project analyzes song duration data using Python and visualizes trends over time.  
It processes raw duration values, converts them into seconds, and identifies the longest and shortest songs in the dataset.

---

## 🚀 Features

- Load and explore song dataset using Pandas
- Identify:
  - Longest song
  - Shortest song
- Convert duration from `hh:mm:ss` format to total seconds
- Generate a time-series plot of song duration vs year
- Basic data inspection and cleaning

---

## 🛠️ Tech Stack

- Python 3
- Jupyter Notebook
- Libraries:
  - pandas
  - matplotlib
